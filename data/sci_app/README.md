# sci_app 数据层

## 数据库分层概述

sci_app（Application System Layer）是应用系统层，用于存储应用系统所需的用户管理、菜单权限和系统配置信息。独立于数据仓库分层架构，单独维护。

**核心职能：**
- 存储应用系统所需的用户管理信息
- 提供菜单结构和权限控制功能
- 管理系统配置和参数设置
- 支持审计日志和操作记录

**设计原则：**
- 支持灵活的菜单层级结构
- 实现基于角色的访问控制（RBAC）
- 记录关键业务操作的审计日志
- 与数据仓库分层隔离，单独维护

## 1. 数据表快速导航

sci_app层用于存储应用系统表，目前无预定义表结构。

在实际使用中，可根据业务需求创建系统表，常见类型包括：

| 数据表类型 | 说明 |
| ---------- | ---- |
| 用户信息表 | 存储用户基本信息 |
| 菜单表 | 存储系统菜单结构 |
| 角色表 | 存储角色定义 |
| 权限表 | 存储权限配置 |
| 用户角色关联表 | 用户与角色关系 |
