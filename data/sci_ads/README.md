# sci_ads 数据层

## 数据库分层概述

sci_ads（Application Data Store Layer）是应用数据存储层，用于存储面向业务应用的数据分析结果和报表。ADS层是数据仓库体系的最顶层，直接对接业务系统的数据消费需求。

**核心职能：**
- 存储面向应用的数据分析结果
- 提供业务报表和仪表盘数据支持
- 实现KPI指标和度量的预计算
- 支持实时和准实时的数据服务

**设计原则：**
- 采用星型或雪花模型进行数据建模
- 优化查询性能支持快速报表生成
- 数据按业务主题域进行组织划分
- 支持历史数据分析对比需求

## 1. 数据表快速导航

sci_ads层用于面向业务应用的数据分析结果存储，目前无预定义表结构。

在实际使用中，可根据业务需求创建分析结果表，常见类型包括：

| 应用表类型 | 说明 |
| ---------- | ---- |
| 风险预警表 | 存储触发阈值告警的风险记录 |
| 绩效报表表 | 汇总供应商、工厂等维度的绩效指标 |
| 影响分析表 | 分析关税、政策变化对业务的影响 |
| KPI汇总表 | 存储关键绩效指标的预计算结果 |
| Dashboard数据表 | 为可视化仪表盘提供数据支撑 |
